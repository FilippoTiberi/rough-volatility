This repository contains a lightweight implementation of key building blocks
for **rough volatility** modelling:

- Simulation of **fractional Brownian motion (fBm)**
- **Hurst exponent estimation**
- A toy implementation of the **rough Bergomi** model
- Monte Carlo option pricing under rough Bergomi

The focus is educational/quant-research oriented, not production-level code.
